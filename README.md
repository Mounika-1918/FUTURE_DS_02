# Customer Retention & Churn Analysis
## Future Interns - Data Science & Analytics Internship
### Task 2 | CIN: FIT/MAR26/DS14855

##  Project Overview
Analyzed customer churn data to identify retention patterns, 
key churn drivers, and actionable recommendations for a 
subscription-based business platform.

##  Tools Used
- Python (Google Colab) - Data Cleaning & Analysis
- Tableau Public - Interactive Dashboard

##  Dataset
- Customer Churn Dataset (Kaggle)
- 64,374 records | 12 features

##  Key Insights
-  Overall Churn Rate: 47.4% — critically high!
-  Payment Delay = #1 churn predictor (76.6% churn!)
-  5+ Support Calls = 60% churn rate
-  2-Year mark = Critical churn spike
-  Revenue Lost: $15.8M

##  Top Recommendations
1. Auto-pay reminders & flexible payment options
2. Improve support quality & response time
3. Quarterly/Annual plan incentives
4. 2-Year loyalty rewards program
5. Senior-friendly UX improvements
